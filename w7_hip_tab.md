## Hip Orthopaedic Conditions — Revision Table

| AVN of hip | Detail |
|---|---|
| Definition | ↓ blood flow → femoral head death → collapse |
| Demographics | Male>female, 35–50yo, **bilateral 80%** |
| Direct causes | Trauma (medial femoral circumflex), irradiation, sickle cell, Caisson, Gaucher |
| Indirect causes | **Alcohol, steroids, SLE**, hypercoagulable, HIV |
| Pathophysiology | Coagulation → venous thrombosis → ↑ intraosseous pressure → collapse |
| Symptoms | Insidious **anterior hip pain**, worse stairs/impact |
| Imaging | **MRI 99% sens/spec**; T1 dark band, T2 bright (oedema) |

| Steinberg/Ficat stage | Finding |
|---|---|
| 0–I | Normal X-ray; abnormal MRI |
| II | Cyst / sclerosis |
| **III** | **Crescent sign (subchondral collapse)** |
| IV | Head flattening |
| V | Joint narrowing |
| VI | Advanced degeneration |

| AVN treatment | When |
|---|---|
| Bisphosphonates | Pre-collapse (0–II) |
| **Core decompression** | Early, pre-collapse, reversible |
| Rotational osteotomy | Small lesion <15% |
| Free-fibula transfer | Young, pre/post collapse |
| **THR** | Crescent/collapse, >40, irreversible cause |

| Hip OA | Detail |
|---|---|
| Risk | Female>male, trauma, dysplasia, SCFE, Perthes |
| Pathology | ↓ proteoglycans, MMP/TIMP imbalance, IL-1/6/TNF-α |
| Symptoms | **Groin pain**, ↓ internal rotation, stiffness |
| X-ray | Narrowing, osteophytes, sclerosis, cysts |

| Tonnis grade | Finding |
|---|---|
| 0 | Normal |
| 1 | Sclerosis + slight narrowing |
| 2 | Large cysts + severe narrowing |
| 3 | Severe head deformity |

| OA treatment | |
|---|---|
| First line | NSAID/tramadol, **stick in opposite hand**, weight loss, PT, steroid injection |
| Operative | PAO (young dysplasia), resurfacing (young male), **THA (>50, end-stage)** |

| Total Hip Replacement | Detail |
|---|---|
| **Charnley** | Low-friction: metal head + poly cup + cement |
| Survival | ~80% at 25 years |
| **Chief contraindication** | **Sepsis (overt/latent)** |
| Fixation | Press-fit (biologic) vs cemented (grout) |
| Bearing | Metal/ceramic/titanium on polyethylene |
| Acute complications | Infection, dislocation, sciatic palsy, periprosthetic fracture, LLD |
| Chronic complications | **Aseptic loosening**, HO, impingement |

**Bold = high-yield.**
