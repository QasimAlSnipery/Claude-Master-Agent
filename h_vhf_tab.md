## Viral Haemorrhagic Fever / CCHF — Revision Table

| VHF overview | Detail |
|---|---|
| Mechanism | Vessel damage → leak + bleeding |
| EMR examples | **CCHF**, yellow fever, dengue, Ebola, Rift Valley |

| CCHF | Detail |
|---|---|
| Virus | Nairovirus, **Bunyaviridae** |
| **Vector** | **Hyalomma tick** |
| Hosts | Cattle, sheep, goats |
| **CFR** | **10–40%** |
| Region | Middle East, Africa, Balkans, Asia |

| Transmission | Route |
|---|---|
| Primary | Tick bite / **animal blood at slaughter** |
| Who | Livestock, abattoir workers, vets |
| Also | **Human-to-human**, nosocomial |

| Incubation | Time |
|---|---|
| Tick bite | 1–3 d (max 9) |
| Blood/tissue | 5–6 d (max 13) |

| Clinical | Feature |
|---|---|
| Onset | Sudden fever, myalgia, photophobia |
| Day 2–4 | RUQ pain, hepatomegaly, **petechiae** |
| Severe | Ecchymoses, hepatitis, multi-organ failure |

| Diagnosis | ELISA, antigen, **RT-PCR**, culture |
|---|---|

| Treatment | **Supportive + ribavirin** |
|---|---|

| Prevention | Detail |
|---|---|
| Animals | Acaricides, 14-d quarantine (no vaccine) |
| Tick→human | Light clothing, repellent, check skin |
| Animal→human | Gloves/PPE at slaughter |
| Human→human | PPE, hand hygiene |

**Bold = high-yield.**
