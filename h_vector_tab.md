## Vector-Borne Infections — Revision Table

| Transmission | Detail |
|---|---|
| **Biological** | Pathogen **multiplies in vector** (malaria, dengue) |
| Mechanical | Physical carry, no multiplication (housefly) |

| Vector | Disease (agent) |
|---|---|
| **Anopheles mosquito** | **Malaria** (Plasmodium) |
| **Aedes mosquito** | Dengue, **Yellow fever**, Chikungunya, Zika |
| Culex mosquito | Encephalitis / West Nile |
| **Sandfly (phlebotomus)** | Leishmania |
| Triatomine bug | Chagas (T. cruzi) |
| **Rodent flea** | Plague (Yersinia pestis) |
| Lice / flea | Typhus (Rickettsia) |
| Dog tick | Rocky Mountain fever |
| Housefly | Cholera, dysentery, typhoid (food) |
| Snail | Schistosomiasis (skin penetration) |

| Yellow fever | Detail |
|---|---|
| Type | Viral haemorrhagic (Aedes), jaundice |
| **Vaccine** | **Single dose = lifelong**, not <9 mo |

| Malaria | Detail |
|---|---|
| Burden | 249M cases, 608k deaths; **<5 = 80% deaths** |
| Species | vivax, **falciparum** (deadly), malariae, ovale |
| **Diagnosis** | **Blood film (thick+thin)**, RDT |
| Prophylaxis | Travellers, soldiers (short-term) |
| Vaccines | RTS,S/AS01; R21 |

| Control program | 1 Recognise → 2 Notify → 3 Source → 4 Extent |
|---|---|

| Outbreak aim | ↓ primary + secondary cases, prevent recurrence |
|---|---|

**Bold = high-yield.**
