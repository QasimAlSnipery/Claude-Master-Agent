## Rabies — Revision Table

| Basics | Detail |
|---|---|
| Virus | **Lyssavirus, Rhabdoviridae** (bullet RNA) |
| Transmission | Bite/lick/scratch (dog, bat) |
| Spread | Muscle → **nerves → CNS** → salivary glands |
| Not in | Blood |
| **Pathognomonic** | **Negri bodies** (absent 20%) |

| Incubation | 20–90 d (shorter: head bite, child, high dose) |
|---|---|

| Phase | Feature |
|---|---|
| Prodrome (2–10d) | Fever, **wound paraesthesia** |
| **Furious (80%)** | **Hydrophobia, aerophobia**, agitation, lucid intervals |
| Dumb (20%) | Flaccid paralysis, no hydrophobia |
| End | Coma → death |

| Diagnosis | Detail |
|---|---|
| **DFA** | Gold standard (antigen) |
| CSF antibody | Early, infection only |
| **RT-PCR** | Most sensitive |
| Negri bodies | Postmortem |

| PEP (life-saving) | Step |
|---|---|
| 1 | **Wash soap+water 15 min** (no suture) |
| 2 | **Vaccine** (HDCV) IM |
| 3 | **RIG (HRIG 20 IU/kg)** around wound |
| 4 | Tetanus + antibiotics |

| Pre-exposure | Vaccine days 0, 7, 21/28 |
|---|---|

| Prognosis | ~100% fatal once symptomatic |
|---|---|

**Bold = high-yield.**
