# PREMEDICATION IN ANAESTHESIA (Revision Table)
Bold = high-yield. Dr Gona Ahmed.

## GOALS &amp; DRUG CLASSES
| Topic | Key points |
|---|---|
| Goals | Anxiolysis/sedation, haemodynamic stability, **↓gastric volume/acidity**, ↓anaesthetic needs, analgesia, **prevent PONV** |
| Classes | Benzodiazepines, opioids, anticholinergics, antiemetics, H2/PPI/antacid, alpha-2/beta-blocker, NSAIDs |

## BENZODIAZEPINES
| Aspect | Key points |
|---|---|
| Mechanism | **Enhance GABA at GABA-A → ↑Cl⁻ → hyperpolarization** (sedation, amnesia, anxiolysis) |
| **Midazolam** | 1–2mg IV; rapid, short; **very strong amnesia**; **paediatric gold standard** |
| Diazepam/Lorazepam | Diazepam (moderate amnesia 2–4h); lorazepam (strong, 6–8h) |
| Side effects | **Respiratory depression (with opioids)**, hypotension, paradoxical reaction; avoid in pregnancy |

## ANALGESIA &amp; OTHER
| Class | Key points |
|---|---|
| Opioids | Fentanyl 1–2mcg/kg; morphine 0.1–0.2mg/kg (severe) |
| Non-opioid | Paracetamol 1g; **gabapentinoids (Ca channel block)**; NSAIDs; ketamine (NMDA); clonidine/dexmedetomidine (alpha-2) |
| Anticholinergics | Atropine/glycopyrrolate/scopolamine — ↓secretions, prevent bradycardia |
| **Antiemetics** | **5-HT3 (ondansetron) = gold standard**; metoclopramide (↓aspiration); dexamethasone |
| Acid suppression | H2/PPI (↓acid + volume); **sodium citrate antacid (↑pH not volume) — obstetrics/trauma** |
| Local anaesthetics | Na⁺ block; lidocaine (blunt propofol pain), bupivacaine; **EMLA for paediatric cannulation** |
