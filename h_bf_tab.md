## Breastfeeding — Revision Table

| Recommendation | Detail |
|---|---|
| Exclusive | **0–6 months** (no other food/drink) |
| Continue | To **2 years** + complementary |
| Initiate | **Within 1 hour** of birth |

| Hormone | Role |
|---|---|
| **Prolactin** (ant. pituitary) | Milk production |
| **Oxytocin** (post. pituitary) | Milk ejection (let-down) |
| Oxytocin ↑ | Suckling, relaxation, seeing baby |
| Oxytocin ↓ | Stress, pain, fatigue, smoking |

| Composition | % |
|---|---|
| Water | 87% |
| Lactose | 7% |
| Fat | 4% |
| Protein | 1% |

| Phase | Detail |
|---|---|
| **Colostrum** (wk 1) | Thick yellow, **immunoglobulins ("1st vaccine")** |
| Transitional (7–14d) | More fat/lactose |
| Mature (>2wk) | 65–75 kcal/100mL |
| Fore-milk | Water, protein |
| **Hind-milk** | **High fat → growth** |

| Attachment signs | Mouth wide, chin to breast, lower lip out, more areola above |
|---|---|

| Reflexes | Rooting, sucking, swallowing |
|---|---|

| Mother benefits | ↓ bleeding, ↓ breast/ovarian cancer, contraception, bonding |
|---|---|

**Bold = high-yield.**
