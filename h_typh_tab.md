## Typhoid Fever — Revision Table

| Basics | Detail |
|---|---|
| Organism | **Salmonella typhi** (G-ve bacillus) |
| Antigens | H (flagellar), O (somatic), **Vi (virulence)** |
| Reservoir | **Humans only** |
| Spread | Faecal-oral food/water, carriers |
| Incubation | 6–30 days |

| Course | Feature |
|---|---|
| Week 1 | **Step-ladder fever**, malaise |
| Week 2–3 | Fastigium: pea-soup diarrhoea, typhoid state, **perforation** |
| Week 4 | Defervescence |
| Week 5 | Convalescence, **relapse possible** |

| Classic signs | |
|---|---|
| **Relative bradycardia** | Fever w/o tachycardia |
| **Rose spots** | Trunk, fade on pressure |
| Organomegaly | Spleno + hepatomegaly |

| Complication | Note |
|---|---|
| **Intestinal perforation/haemorrhage** | **Most serious** (wk 2–3) |
| Other | Myocarditis, pneumonia, meningitis |

| Diagnosis | Timing |
|---|---|
| **Blood culture** | 90% week 1 → 25% week 4 |
| Bone marrow culture | Most sensitive |
| Stool/urine | Weeks 3–4 |
| Duodenal string | **Carriers** |
| Widal | O & H Ab; weak sensitivity/specificity |

| Treatment | Detail |
|---|---|
| First line | **Ciprofloxacin 15 mg/kg × 7d** |
| Resistant | Azithromycin / ceftriaxone |
| Severe | Dexamethasone |
| Carrier | Co-trimox 6wk / cipro 4wk; **3 neg stools** |

| Vaccine | Detail |
|---|---|
| Ty21a | Oral live, booster 5yr, not pregnancy |
| Typhim-Vi | Polysaccharide IM, booster 2yr |
| TCV | Conjugate, usable <2yr |

**Bold = high-yield.**
