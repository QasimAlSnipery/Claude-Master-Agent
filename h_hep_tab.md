## Viral Hepatitis A–E — Revision Table

| Virus | Spread | Chronic | Prevention |
|---|---|---|---|
| **A** | Faecal-oral | No | Vaccine + Ig |
| **B** | Blood/sexual/perinatal | **Yes** | Vaccine + HBIG |
| **C** | Blood | **70–85%** | No vaccine; **cure (DAA)** |
| **D** | Blood (**needs HBV**) | Yes | HBV vaccine |
| **E** | Faecal-oral | No | Safe water |

| Virus | Incubation | Note |
|---|---|---|
| A | ~30 d | Jaundice ↑ with age; IgM = acute |
| B | 60–90 d | Chronicity ↑ if younger |
| C | 6–7 wk | Mostly chronic, genotype 1 |
| E | ~40 d | **20% mortality in pregnancy** |

| HBV serology | Meaning |
|---|---|
| **HBsAg** | Infection (>6mo = chronic) |
| **Anti-HBs** | Recovery / immunity / vaccine |
| Anti-HBc IgM | Acute |
| Anti-HBc IgG | Past/chronic |
| **HBeAg** | Active replication / infective |
| HBV-DNA | Replication (monitor Rx) |

| HBV chronicity by age | % |
|---|---|
| Perinatal | 30–90% |
| Adult | 2–10% |

| Treatment | Detail |
|---|---|
| HBV | Tenofovir, entecavir (not cure) |
| **HCV** | **DAAs (sofosbuvir...) cure SVR>95%** |
| Acute (any) | Supportive |

| HBV vaccine | **3 doses (0,1,6), IM deltoid** |
|---|---|
| HBIG | Post-exposure <48h, neonates of HBeAg+ |

| Labs | ALT>AST; **PT = prognosis**; albumin normal |
|---|---|

**Bold = high-yield.**
