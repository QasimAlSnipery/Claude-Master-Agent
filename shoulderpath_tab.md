# COMMON SHOULDER &amp; UPPER LIMB PATHOLOGY — Revision Table
Bold = high-yield. Dr Saywan Asaad.

## SHOULDER &amp; ELBOW SOFT TISSUE
| Condition | Clinical / tests | Management |
|---|---|---|
| **Impingement** | **Painful arc 60–120°**, night pain; **Neer, Hawkins-Kennedy, empty can** | NSAIDs, physio, subacromial steroid → **arthroscopic decompression** |
| **Frozen shoulder** | **Loss of passive ER = hallmark**; **DM assoc**; freezing/frozen/thawing | NSAIDs, physio, **intra-articular steroid (early)**, MUA if refractory |
| **Calcific tendinitis** | Hydroxyapatite in supraspinatus; **resorptive phase = severe pain**; X-ray dense calcification | NSAIDs, **barbotage, shock-wave**; arthroscopic removal |
| **Tennis elbow** | **ECRB**; lateral pain, resisted wrist extension; **Cozen's/Mill's/Maudsley** | Rest, eccentric strengthening, brace, PRP; surgery if &gt;6–12mo |

## CONGENITAL UPPER LIMB
| Deformity | Key points |
|---|---|
| Radial longitudinal deficiency | **VACTERL, Holt-Oram, Fanconi**; Bayne I–IV; **pollicization** |
| Syndactyly | Failed separation; **Apert, Poland**; release 6–12mo |
| Polydactyly | Extra digits; Ellis-van Creveld, Patau; excision (Wassel for thumb) |
| Thumb hypoplasia | Holt-Oram, VACTERL; Blauth I–V; opponensplasty/pollicization |
| Macrodactyly | **Proteus, NF1**; debulking, epiphysiodesis |
