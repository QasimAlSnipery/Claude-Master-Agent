## Air-Borne Diseases (Diphtheria & Mumps) — Revision Table

| Transmission route | Reach |
|---|---|
| **Droplets** | Large, short range, **upper tract** |
| **Droplet nuclei** | Small, far, **lower tract** |
| Dust | Reservoir |

| Control | Method |
|---|---|
| Agent | Eliminate reservoir, disinfect |
| Transmission | Ventilation, UV, masks, no crowding |
| Host | Immunisation, Ig, chemoprophylaxis |

| Diphtheria | Detail |
|---|---|
| Organism | **C. diphtheriae** + cytotoxin |
| Hallmark | **Pseudomembrane**, bull neck |
| Toxin effects | **Myocarditis, heart block**, neuropathy |
| CFR | 5–10% |
| Incubation | 2–5 days |
| Prevention | **Toxoid (DTP/PENTA)** |
| Treatment | **Antitoxin ASAP** + antibiotics |
| Contacts | Benzathine penicillin + booster |

| Mumps | Detail |
|---|---|
| Virus | Paramyxovirus |
| Incubation | 14–24 days |
| Clinical | **Parotid swelling**, 1/3 asymptomatic |
| Complications | **Orchitis** (♂), meningitis, pancreatitis |
| Diagnosis | Clinical, IgM |
| Treatment | Supportive (viral) |
| Prevention | **MMR** (2 doses); CI pregnancy |

**Bold = high-yield.**
