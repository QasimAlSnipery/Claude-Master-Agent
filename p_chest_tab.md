## Chest Wall Deformities — Revision Table

| Scoliosis | Detail |
|---|---|
| Definition | Lateral curve + **rotation** of spine |
| Commonest | Idiopathic; thoracic **right**, girls |
| **Cobb's angle** | 0–10° normal; **>10° scoliosis** |
| Lung effect | **Restrictive** (↓VC/TLC/DLCO), normal FEV1/FVC |
| Severe ≥100° | Sleep desaturation, **PHT → cor pulmonale** |
| Death | **Respiratory failure** |
| Treatment | Fusion; growth rods (children) |

| Pectus excavatum | Detail |
|---|---|
| Definition | **Depressed sternum** (funnel), commonest |
| Sex | **3:1 male**; Marfan link |
| **Haller index** | Normal 2.5; **>3.25 significant** |
| Symptoms | Mostly cosmetic; MVP, murmur |
| Surgery timing | **10–14 yrs** |
| **Nuss** | Minimally invasive bar (2–4 yr) — commonest |
| **Ravitch** | Open: cartilage resection + osteotomy |

| Pectus carinatum | Detail |
|---|---|
| Definition | **Prominent (keel) sternum** |
| Sex | **M:F 4:1**, less common |
| Cardiac | 1–3% anomaly → **echo** |
| Treatment | Ravitch or **bracing** |

| Poland syndrome | Detail |
|---|---|
| Triad | **Absent pectoralis + syndactyly** |
| CXR | **Hyperlucent lung** |
| Complaint | Cosmetic |
