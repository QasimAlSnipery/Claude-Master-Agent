## Control of Diarrhoeal Disease — Revision Table

| Concept | Detail |
|---|---|
| Diarrhoea | **≥3 watery stools/24h**; consistency > frequency |
| Acute / persistent | ≤14d / ≥14d |
| Dysentery | **Blood + mucus** (amoebic/bacillary) |
| Causes | Viruses, bacteria, protozoa (not fungi) |
| The killer | **Dehydration** |

| Dehydration | A (none) | B (some) | C (severe) |
|---|---|---|---|
| Condition | Alert | Restless | **Lethargic** |
| Thirst | Normal | Eager | Can't drink |
| Skin pinch | Quick | Slow | **Very slow** |
| Plan | Home | PHC | Hospital |

| Plan | Action |
|---|---|
| **A** | ORS + feeding + teach mother (home) |
| **B** | **ORS 75 mL/kg over 4 h** (PHC) |
| **C** | IV Ringer lactate **100 mL/kg** (hospital) |

| Plan C IV schedule | First | Then |
|---|---|---|
| **<12 mo** | 30 mL/kg / 1 h | 70 mL/kg / 5 h |
| **≥12 mo** | 30 mL/kg / 30 min | 70 mL/kg / 2.5 h |

| ORS (per 1 L) | Amount |
|---|---|
| Na | 3.5 g |
| NaHCO₃ | 2.2 g |
| KCl | 1.5 g |
| **Glucose** | 20 g (drives Na co-transport) |

| Rule | Detail |
|---|---|
| <6 mo + dehydrated | **Hospital** |
| Overhydration sign | **Puffy face/eyes** → slow |
| Contraindicated <5 | **Anti-motility** (paralytic ileus) |

**Bold = high-yield.**
