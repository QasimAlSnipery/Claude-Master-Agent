## Liver & Hepatobiliary Imaging — Revision Table

| Modality | Normal liver |
|---|---|
| US | Uniform, bright portal triads |
| CT | **65 ± 5 HU**, > muscle/spleen |
| MRI | **> spleen on T1**, < on T2 |

| Lesion class | DDx |
|---|---|
| Cystic | Simple cyst, hydatid, cystic mets |
| Complex | Abscess, necrotic tumour, haematoma |
| Solid | Mets, HCC, haemangioma |

| Metastases | Imaging |
|---|---|
| Pattern | **Multiple, peripheral, variable** |
| US | Usually hypoechoic / target |
| CT | Ring enhancement, lower density |
| MRI | **Low T1, high T2** |

| Benign lesion | Signature |
|---|---|
| Simple cyst | **0 HU**, echo-free, posterior enhancement |
| Hydatid | **Wall calcification + daughter cysts** |
| **Haemangioma** | **Centripetal fill-in + T2 light-bulb** |
| Abscess | Thick irregular wall + debris, aspirate |

| Primary malignant | Large, usually solitary (HCC, cholangio) |
|---|---|

| Other | Finding |
|---|---|
| **Trauma** | **CT best**; contusion/laceration + subcapsular haematoma |
| Cirrhosis | RT lobe atrophy, irregular outline |
| **Portal HTN** | Cirrhosis commonest; splenomegaly, collaterals, **TIPS** |
| Fatty liver | Bright US, low CT, **high T1 MRI** |

**Bold = high-yield.**
