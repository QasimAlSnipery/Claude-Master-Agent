## Regional Anaesthesia — Revision Table

| Technique | Use |
|---|---|
| Spinal | Abdomen, pelvis, lower limb |
| Epidural | Neck → lower limb; post-op analgesia |
| Caudal | Perineal |
| IV regional (Bier) | Upper/lower limb |
| Interscalene block | Shoulder, elbow |
| **Sciatic / femoral** | Leg |
| **Spinal/epidural** | **Caesarean (choice)** |

| Neuraxial CI | Type |
|---|---|
| Patient refusal | Absolute |
| **Anticoagulation** | Absolute |
| Local sepsis | Absolute |
| Raised ICP | Absolute |
| Severe valve stenosis, hypovolaemia | Relative |
| Previous back surgery, neuro disease | Relative |

| Neuraxial complication | Note |
|---|---|
| Hypotension / bradycardia | Can → arrest (mid-thoracic) |
| High/total spinal | Compromises breathing |
| PDPH | Headache |
| Spinal haematoma | Coagulopathy → paraplegia |
| Serious (death/paraplegia) | 1:50,000–1:140,000 |

| LAST | Detail |
|---|---|
| Cause | Excess LA in circulation |
| Early signs | Peri-oral tingling, metallic taste, tinnitus |
| Late | Convulsions, CVS collapse, resp arrest |
| **Treatment** | Stop, ABC/O₂, seizures Rx, **20% lipid emulsion** |

| Agent | Class / dose |
|---|---|
| **Lidocaine** | Amide, **2 mg/kg**, medium-acting |
| **Bupivacaine** | Amide, **3 mg/kg (7+adrenaline)**, cardiotoxic |
| Allergy | Esters > amides |

| Epinephrine (1:200,000) | |
|---|---|
| Advantages | ↓ absorption, prolongs block, test dose |
| **Avoid** | Digits/penis, unstable angina, IV regional, dysrhythmia |

**Bold = high-yield.**
