# SURGICAL AUDIT &amp; RESEARCH — Revision Table
Bold = high-yield. Prof. Kamal Saeed.

## CLINICAL AUDIT
| Topic | Key points |
|---|---|
| Definition | **Quality improvement — systematic review against explicit criteria** (NICE 2002); "audire" = to hear |
| Pioneer | **Florence Nightingale** (Crimea) → mortality 40% → 2% |
| **Care aspects** | **Structure (what's in place), Process (what you do), Outcome (results)** |
| Change levels | Individual, instrument, team, institution, regional, national |
| **Audit cycle** | **Question → standards → measure → compare → gap analysis → change → RE-AUDIT** |

## RESEARCH
| Topic | Key points |
|---|---|
| Definition | Systematic investigation for new facts using scientifically sound methods |
| **Audit ≠ research** | Audit is NOT research |
| Study types | Observational (retro/prospective), **case-control, cross-sectional, longitudinal, RCT** |
| Identification | New knowledge; RCTs; **always needs ethics/trust approval**; powered sample; hypothesis-driven; publishable |

## RESEARCH vs AUDIT
| | Research | Audit |
|---|---|---|
| Question | **"Are we singing the RIGHT song?"** | **"Are we singing this song RIGHT?"** |
| Purpose | New knowledge / set standards | Measure vs existing standards |
| Ethics approval | **Always required** | Not required |
