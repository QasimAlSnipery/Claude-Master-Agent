# COMMON ANKLE &amp; FOOT DEFORMITIES (Revision Table)
Bold = high-yield. Prof. Omer Barawi.

## TERMINOLOGY
| Term | Meaning |
|---|---|
| Plantigrade | Normal (sole ⊥ leg) |
| Equinus / Calcaneus | Hindfoot plantarflexion / fixed dorsiflexion |
| Rocker-bottom | Mid-foot dorsiflexion (vertical talus) |
| Planus / Cavus | Flat / high arch |
| Hallux valgus / hammer / claw | Lateral big toe / PIP flexion / IP flexion + MTP hyperextension |

## CLUBFOOT (Talipes Equinovarus)
| Aspect | Key points |
|---|---|
| Deformity | **Heel equinus + hindfoot varus + forefoot adduction + supination** |
| Epidemiology | **1–2/1000; boys 2:1; bilateral 1/3** |
| Causes | Idiopathic (commonest); spina bifida, CP, polio, arthrogryposis |
| Clinical | **True = big toe CANNOT reach shin; postural = CAN reach (resolves)**; examine spine |
| **Ponseti method** | Start 1–2 days of birth; **correct all EXCEPT equinus → equinus LAST (avoid rocker-bottom)**; weekly casts ×5 + **Achilles tenotomy** → brace; relapse common |
| Resistant/relapsed | Surgery / **Ilizarov** |

## HALLUX VALGUS
| Aspect | Key points |
|---|---|
| Definition | **Lateral big toe deviation; commonest foot deformity**; metatarsus primus varus |
| Clinical | Pain over **bunion** (1st MT head, medial); great toe valgus |
| X-ray (standing) | **Intermetatarsal angle &lt;9°, HV angle &lt;15° normal** |
| Treatment | **Flat shoes**; surgery if painful; **not for cosmesis** (recurrence + pain transfer) |
