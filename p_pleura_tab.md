## Pleural Diseases — Revision Table

| Anatomy | Fact |
|---|---|
| Visceral pleura | Lung + fissures, **painless** (phrenic) |
| Parietal pleura | Chest wall, **somatic pain** (intercostal) |
| Normal fluid | ~0.25 mL/kg, Starling forces |

| Pneumothorax | Detail |
|---|---|
| PSP | Apical **blebs**, tall thin young male smoker |
| SSP | COPD, TB, CF, fibrosis |
| **Tension** | Decompress **2nd ICS MCL** → tube (clinical dx) |
| Sign | Deep sulcus (supine), hyperresonance |
| Diagnosis | Upright **CXR** |
| Small | Observe + O₂ (2.2%/day, 79%) |
| Recurrence | **30% after first** (<2 yrs) |
| Surgery | **VATS bullectomy >95%**; leak>5d, recurrent |

| Empyema | Detail |
|---|---|
| Definition | **Pus in pleural cavity** |
| Stages | Exudative → **fibropurulent (loculations)** → organising (**peel**) |
| Light's exudate | Protein>0.5, LDH>0.6, **pH<7.0** |
| Treatment | Tube (32–38Fr) + fibrinolytics; VATS / decortication |

| Malignant effusion | Detail |
|---|---|
| Frequency | 22%; **95% metastatic** (lung+breast 75%) |
| Type | Exudate |
| CT | Nodular pleural thickening |
| Treatment | Drain + **pleurodesis** (talc/bleomycin) |

| Chylothorax | Chyle in pleura; drain + thoracic duct ligation |
|---|---|

| Mesothelioma (MPM) | Detail |
|---|---|
| Cause | **Asbestos >80%**, 20-yr latency, 5:1 male |
| Diagnosis | **Thoracoscopic biopsy (gold standard)** |
| Surgery | Extrapleural pneumonectomy (selected) |

**Bold = high-yield.**
