## Leishmaniasis — Revision Table

| Basics | Detail |
|---|---|
| Parasite | **Leishmania** (intracellular protozoan) |
| **Vector** | **Female sandfly (Phlebotomus)** |
| Stages | Promastigote (injected) → amastigote (macrophage) |

| Form | Species | Note |
|---|---|---|
| **Cutaneous** | L. tropica, L. major | **Baghdad boil** |
| Mucocutaneous | L. braziliensis | Nasopharyngeal |
| **Visceral** | **L. donovani** | **Kala-azar** |

| Cutaneous | Detail |
|---|---|
| Site | Exposed skin (face, limbs) |
| Lesion | Macule→papule→**ulcer**→scar |
| Diagnosis | **Giemsa smear (active margin)**, NNN culture |
| Treatment | Paromomycin, miltefosine, antimonials |

| Visceral (kala-azar) | Detail |
|---|---|
| Spread | **Spleen, liver, bone marrow** |
| Features | **Hepatosplenomegaly, pancytopenia, fever** |
| Diagnosis | **Bone marrow (60–85%)**, splenic (>95%), PCR |
| Treatment | **Amphotericin B (first-line)**, miltefosine |
| Untreated | Fatal |

| Iraq | CL across Iraq (not Kurdistan 3 govs) |
|---|---|

| Prevention | Vector control, ITN, repellent, early Rx |
|---|---|

**Bold = high-yield.**
