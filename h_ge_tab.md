## Gastroenteritis — Revision Table

| Definitions | Detail |
|---|---|
| Gastroenteritis | GI inflammation (infx/non-infx) |
| Diarrhoea | **≥3 loose stools/24h**, acute <14d |
| Main danger | **Dehydration** |

| Cause | Examples |
|---|---|
| **Viral (~70%)** | **Norovirus** (adults), **rotavirus** (children) |
| Bacterial (10–20%) | E. coli O157, Salmonella, Shigella, Campylobacter |
| Parasitic (<10%) | Giardia, Cryptosporidium, Entamoeba |
| Non-infectious | NSAIDs, antibiotics, lactose/gluten |

| Pathophysiology | Organisms |
|---|---|
| Watery (toxin) | S. aureus, B. cereus, **ETEC, cholera** |
| **Bloody (invasive)** | **Salmonella, Shigella, Campylobacter, STEC** |

| Treatment | Detail |
|---|---|
| **Rehydration** | ORS / IV (most crucial) |
| Diet | BRAT; avoid fat/caffeine |
| Antidiarrhoeal | **NOT in infectious GE** |
| Antibiotic | Only confirmed bacterial (cipro) |
| Probiotic | Recovery |

| Prevention | Detail |
|---|---|
| Hygiene | **Handwashing** |
| Water/food | Safe water, cook, separate raw meat |
| **Rotavirus vaccine** | RotaTeq (2,4,6mo) / Rotarix (2,4mo), oral |

**Bold = high-yield.**
