## Thyroid Diseases — Revision Table

| Axis | Fact |
|---|---|
| Gland | Butterfly, 10–20 g, T4 + T3 |
| Control | **TSH** (ant. pituitary), TRH (hypothalamus) |
| Calcitonin | C-cells, lowers Ca |

| TFT pattern | T3 | T4 | TSH |
|---|---|---|---|
| **Hyperthyroid** | ↑ | ↑ | **↓** |
| **Hypothyroid** | ↓ | ↓ | **↑** |

| Hyperthyroidism | Detail |
|---|---|
| Symptoms | Weight loss, heat intolerance, tachycardia, tremor |
| Primary causes | **Graves**, toxic MNG, toxic adenoma |
| Graves eyes | Lid retraction, **proptosis**, chemosis |
| Secondary | TSH-secreting pituitary adenoma |

| Hypothyroidism | Detail |
|---|---|
| Symptoms | Fatigue, weight gain, constipation, cold, dry skin |
| Primary | **Hashimoto**, I-131, thyroidectomy, iodine def |
| Secondary | Pituitary, hypothalamic |

| Thyroiditis | Acute suppurative, subacute painful/painless, **Hashimoto**, Riedel |
|---|---|

| Investigation | Use |
|---|---|
| TFT | T3/T4/TSH |
| US neck | Structure |
| **FNAC** | Nodule cytology |
| Tc-99m scan | Hot toxic adenoma |

| Thyroid storm | Detail |
|---|---|
| Features | Fever, tachycardia, agitation, coma |
| Triggers | Infection, stress, trauma |
| **Treatment** | β-blocker, antithyroid, iodine, glucocorticoid |

| Thyroidectomy complication | Note |
|---|---|
| **Hypocalcaemia** | **Most common** (parathyroid) |
| RLN injury | Hoarseness |
| Haematoma | **Airway emergency → re-op** |
| Superior laryngeal n. | High notes / voice |

**Bold = high-yield.**
