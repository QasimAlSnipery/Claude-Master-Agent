## Measles & Rubella — Revision Table

| Measles | Detail |
|---|---|
| Virus | Paramyxovirus (RNA), 1 genotype |
| Reservoir | Human; **90% attack rate** |
| Transmission | Droplet; winter |
| Incubation | 10–12 days |
| Infectious | **4 days before–4 after rash** |
| Prodrome | **3 C's**: Cough, Coryza, Conjunctivitis |
| Pathognomonic | **Koplik spots** (buccal) |
| Rash | Maculopapular, hairline → down |

| Measles complication | Note |
|---|---|
| Pneumonia | Commonest death (children) |
| Otitis media, diarrhoea | Common |
| Encephalitis | 1/1000, CFR 15% |
| **SSPE** | ~7 yr later, fatal |

| Measles vaccine | Detail |
|---|---|
| Type | Live attenuated (MMR) |
| Schedule | 9 mo, **MMR 12 & 18 mo** |
| CI | Pregnancy, immunosuppression |

| Rubella | Detail |
|---|---|
| Virus | Togaviridae (RNA) |
| Incubation | 14 days |
| Clinical | Mild fever, **posterior lymphadenopathy**, rash |
| Complication | Arthritis (adult ♀ 70%) |
| **CRS** | **Deafness + cataracts + heart defects** (1st trimester, 85%) |
| Vaccine | Live, MMR, from 12 mo; target ♀ childbearing |

**Bold = high-yield.**
