# SURGICAL ASPECTS OF IBD — Revision Table
Bold = high-yield. Dr Farman Hassan.

## UC vs CROHN'S &amp; SEVERITY
| Feature | Ulcerative colitis | Crohn's |
|---|---|---|
| Extent | **Mucosa + submucosa; continuous from rectum** | **Transmural; skip lesions; mouth→anus** |
| Symptoms | Rectal bleeding, tenesmus, mucus | Variable; perianal disease |
| **Severity (UC)** | **Mild &lt;4 / moderate &gt;4 / severe &gt;6 bloody stools; fulminant &gt;10 → toxic megacolon** | — |
| Extra-intestinal | **Arthritis, sacroiliitis, sclerosing cholangitis, erythema nodosum, pyoderma gangrenosum, uveitis** | Same |
| **Cancer risk** | **↑with duration: 1% (10yr) → 20% (30yr); pancolitis &gt;10yr → screening** | Lower |

## SURGERY
| Topic | Key points |
|---|---|
| **UC surgery** | **Curative (colectomy); lifetime risk ~20%**; indications: fulminant, steroid-dependent, dysplasia/cancer, sclerosing cholangitis |
| UC emergency | **Subtotal colectomy + end-ileostomy** |
| UC elective | Proctocolectomy + ileostomy; **IPAA (ileal pouch-anal anastomosis)** |
| IPAA complications | Pelvic infection (leak), SBO 10–15%, **pouchitis → metronidazole/cipro + probiotics** |
| **Crohn's surgery** | **NOT curative — complications only; minimal resection (avoid short bowel)**; ileocaecal resection; stricturoplasty |
| Perianal Crohn's | Abscess → fistula; **seton + infliximab/adalimumab**; LIFT, advancement flap |
| **VTE** | **IBD = 3× risk; LMWH up to 28 days post-op** |
