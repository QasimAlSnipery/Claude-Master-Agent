# VENOUS DISORDERS (Revision Table)
Bold = high-yield. Dr Dezhin Solai.

## ANATOMY &amp; CHRONIC VENOUS INSUFFICIENCY
| Topic | Key points |
|---|---|
| Anatomy | Valves = bicuspid (absent in pelvic/IVC); deep (all valved), superficial (**GSV, SSV**), perforators (~150); **SFJ + SPJ junctions**; one-way flow superficial→deep→heart; calf muscle pump |
| Pathophysiology | **Valve failure → reflux**; **strong genetic component** |
| Myths | **Pelvic masses → DVT (not varicose); pregnancy UNCOVERS hidden varicose veins** (hormonal) |
| Chronic venous disease | Spectrum (spider veins → ulcers); 60–80%; no gender diff; standing/sitting/aging |
| Clinical | Achy/heavy legs, oedema, itching; **gaiter area: hyperpigmentation, eczema, lipodermatosclerosis, ulcer** |
| Key facts | **Gaiter ulcer = venous until proven otherwise; spider veins = sign not cosmetic; CEAP classification** |
| Diagnosis/Mgmt | **Doppler US = gold standard**; compression socks; **endovenous (laser/RFA/foam) replaced surgery** |

## DEEP VEIN THROMBOSIS
| Topic | Key points |
|---|---|
| Overview | Clot in deep veins; **3rd CV death (after MI, stroke)**; starts in calf; **untreated proximal → 50% PE** |
| Complications | PE, death, **post-thrombotic syndrome (20–50%)** |
| Classification | Location (proximal/distal/mixed); duration (acute ≤14d, subacute 14–28d, chronic &gt;28d); provoked vs unprovoked |
| Risk factors | **Immobility** (surgery, hip/knee replacement, flights), tumour/pregnancy, **thrombophilia (protein C/S, AT3, factor V Leiden)**, cancer, OCP, smoking |
| **Pathophysiology** | **Virchow's triad: vessel damage + turbulent flow + hypercoagulability** |
| Evaluation | **D-dimer (sensitive not specific), Doppler US, Wells score** (0–1 low, ≥2 high) |
| **Treatment** | **UFH IV (initial/standard) → LMWH/DOAC** (rivaroxaban); mechanical: compression, IVC filter |
