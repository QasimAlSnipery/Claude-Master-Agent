# UPPER EXTREMITY FRACTURES PART 1 — Revision Table
Bold = high-yield. Shoulder girdle → elbow. Dr Zaid Al-Bachachy.

## SHOULDER GIRDLE
| Injury | Key points |
|---|---|
| **Clavicle** | Groups: **I middle (commonest), II lateral, III medial**. Surgery: open, skin tenting, neurovascular, **floating shoulder** |
| Scapula | Body/neck conservative; **intra-articular → surgery** |
| Scapulothoracic dissociation | High-energy; subclavian + brachial plexus rupture; **&gt;1cm clavicle distraction**; poor outcome |
| AC joint | **AC lig = AP stability; CC (trapezoid+conoid) = SI stability** |
| Sternoclavicular | Anterior (benign bump) vs **posterior (serious — tracheal/vessel)**; serendipity view |

## SHOULDER &amp; HUMERUS
| Injury | Key points |
|---|---|
| **Anterior dislocation (commonest)** | Fall on hand → **Bankart lesion**; **axillary nerve injury (commonest)** |
| Posterior dislocation | **Fit/convulsion/electric shock** (internal rotation/adduction) |
| Luxatio erecta | Inferior; arm fixed abducted |
| **Proximal humerus** | **Osteoporotic postmenopausal women**; Neer parts; **4-part = AVN risk**; minimally displaced (majority) → sling |
| Paediatric proximal | Salter-Harris; mostly non-operative (remodelling) |

## ELBOW &amp; DISTAL HUMERUS
| Injury | Key points |
|---|---|
| Distal humerus (adult) | **AO: A extra-articular, B unicondylar, C bicondylar**; displaced → ORIF |
| **Radial head** | **Mason I–IV**; missed often; pain on pronation/supination; II → screw |
| Olecranon | Comminuted (blow) vs **transverse (traction) → tension band** |
| Elbow dislocation | **90% posterior**; brachial artery, median/ulnar nerve |
| **Supracondylar (children)** | **Commonest paediatric elbow; 95% posterior; Gartland I–III; brachial artery/median nerve; crossed K-wires; gunstock deformity; fat pad sign** |
| Lateral condyle (children) | 2nd commonest; **non-union/malunion/AVN risk** |
