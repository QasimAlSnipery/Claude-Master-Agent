# UPPER EXTREMITY FRACTURES PART 2 — Revision Table
Bold = high-yield. Humeral shaft → hand. Dr Zaid Al-Bachachy.

## SHAFT &amp; FOREARM
| Injury | Key points |
|---|---|
| **Humeral shaft** | Radial nerve risk; **Holstein-Lewis = distal third spiral + radial nerve palsy (wrist drop)**; surgery if floating elbow, open, post-manipulation palsy |
| Both-bone forearm | **Adults → ORIF nearly all**; children mostly non-op (greenstick) |
| **Monteggia** | **Proximal 1/3 ulna + radial head dislocation** → ORIF (adult) |
| **Galeazzi** | **Distal 1/3 radius + DRUJ injury** → ORIF radius + assess DRUJ |
| TFCC | Ulnar wrist pain, **fovea sign**; MRI; conservative → debridement/repair |

## DISTAL RADIUS &amp; CARPUS
| Injury | Key points |
|---|---|
| **Distal radius (adult)** | **Most common ortho injury**; ORIF if instability, dorsal angulation &gt;5°, intra-articular &gt;2mm, **Barton's/Smith's**, critical corner |
| Distal forearm (child) | Closed reduction + cast; CRPP if unstable; ORIF SH III/IV |
| **Scaphoid** | **75% carpal fractures; waist 65%**; **retrograde supply → proximal pole AVN/non-union**; suspect with normal X-ray → **thumb spica + re-image 12–21d** |
| Lunate/perilunate | High-energy dorsiflexion; **trans-scaphoid perilunate**; urgent reduction |

## HAND
| Injury | Key points |
|---|---|
| Metacarpal | Shaft (rotation → fix); **neck = boxer's (5th MC)**; base stable |
| **Bennett's** | Oblique intra-articular base 1st MC, **unstable**, into CMC (punching) |
| **Rolando's** | Comminuted T/Y base 1st MC |
| **Gamekeeper's/skier's thumb** | **UCL of thumb MCP**; **Stener lesion** (trapped by adductor aponeurosis → won't heal → repair) |
