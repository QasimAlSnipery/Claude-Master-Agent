## Hypercalcaemia & Thyroid/Parathyroid Review — Revision Table

| Hypercalcaemia | Value |
|---|---|
| Normal Ca | **8.5–10.2 mg/dL** |
| Mild-moderate | 12–14 mg/dL (asymptomatic) |
| **Crisis** | **>14 mg/dL** (aggressive Rx) |
| Coma/arrest | >18 mg/dL |
| ECG | Long PR, **short QT** |

| Crisis presentation | Confusion, abdominal pain, vomiting, dehydration, anuria |
|---|---|

| Treatment pillar | Agent |
|---|---|
| ↑ renal excretion | **Saline 200–500mL/h** then **furosemide** |
| ↓ bone release | **Bisphosphonate** (osteoclast inhibit) |
| Bridge | **Calcitonin** (fast, short) |
| Adjunct | Glucocorticoid (vit D excess) |

| Thyroid review | Fact |
|---|---|
| Thyroglossal cyst | Midline; **Sistrunk operation** |
| Superior thyroid artery | From **external carotid** |
| Inferior thyroid artery | From thyrocervical trunk |
| RLN injury | Hoarse (uni) / airway (bilateral) |
| Superior laryngeal n. | Can't hit **high notes** |
| TSH | Directly releases T3/T4 |

| Thyroid cancer | Management |
|---|---|
| FNAB "suspicious" | **Lobectomy** |
| **Papillary** | 80%, radiation/Chernobyl, children |
| Follicular | **Total thyroidectomy** + radioiodine |
| Medullary | Screen **MEN2** (pheo + hyperparathyroid) |

| Post-thyroidectomy emergency | **Wound haematoma** → bedside open + re-op |
|---|---|

**Bold = high-yield.**
