## Tumours of Lung & Bronchi — Revision Table

| Group | Share | Behaviour |
|---|---|---|
| **NSCLC** | 80–85% | Slower, often surgical |
| **SCLC** | 10–15% | Aggressive, early mets, chemo/RT |

| NSCLC subtype | Location | Notes |
|---|---|---|
| **Adenocarcinoma** | Peripheral | **Commonest**, non-smokers, women, younger |
| **Squamous cell** | **Central** | Smoking, cavitation, haemoptysis |
| Large cell / LCNEC | Any | Fast, LCNEC ~ SCLC |

| Cause | Detail |
|---|---|
| **Smoking** | **~90%** of cases |
| Other | Asbestos, radon, IPF, radiation |
| Never-smokers | Up to 20% (adeno) |

| Other tumour | Note |
|---|---|
| Carcinoid | <5%, carcinoid syndrome |
| **Hamartoma** | Commonest benign, **coin lesion** |
| Metastasis | "**Cannonball**" nodules |

| Spread | Detail |
|---|---|
| Distant sites | **Brain, bone, liver, adrenals** |
| Local syndromes | **SVC obstruction, Horner's**, hoarseness |
| Nodes | Hilar → mediastinal → supraclavicular |

| IHC marker | Tumour |
|---|---|
| TTF-1 / Napsin A | Adenocarcinoma |
| **p40 / p63** | Squamous |
| Chromogranin/synaptophysin/CD56 | Neuroendocrine |

| TNM | Component |
|---|---|
| T | Size + local invasion |
| N | N0–N3 nodes |
| M | M1a/b/c distant |

| Management | Detail |
|---|---|
| NSCLC early | **Lobectomy** |
| SCLC | **Chemo + radiotherapy** |
| Screening | Low-dose CT (high-risk smokers) |

**Bold = high-yield.**
