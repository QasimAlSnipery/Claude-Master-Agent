## Lymphedema — Revision Table

| Basics | Detail |
|---|---|
| Definition | Chronic swelling: impaired outflow OR overproduction |
| Lymph content | WBC, triglycerides, debris, water, **protein** |
| Primary epidemiology | 1:6000–10,000; **females 2–20×**, peak 12–16 |

| Primary — onset | Features |
|---|---|
| Congenital (Milroy) | Birth/1st yr; Turner, Klinefelter, T21; bilateral |
| **Praecox (Meige's)** | Puberty; **commonest (94%)**; F:M **10:1**; unilateral foot/calf |
| Tarda | After **35 yrs**; ~10% |

| Primary — anatomy | Features |
|---|---|
| Distal obstruction | ⅓; bilateral, mild, female, good prognosis |
| Proximal obstruction | ½; unilateral, severe, relentless → surgery |

| Secondary (commonest overall) | Detail |
|---|---|
| **Cancer** | Breast (axillary dissection + RT) → arm; gynae/uro → leg |
| **Filariasis** | *W. bancrofti* (90%); tropics; eosinophilia; nocturnal microfilariae → elephantiasis |
| Trauma | Burns, wounds, pregnancy |

| Stage | Finding |
|---|---|
| Latent | Fibrosis, no visible edema |
| Grade 1 | **Pits**, reduced by elevation |
| Grade 2 | **Non-pitting**, fibrosis |
| Grade 3 | Irreversible = **elephantiasis** |

| Signs | Detail |
|---|---|
| **Stemmer's sign** | Can't tent skin at base of 2nd toe (hallmark) |
| Square toes / buffalo hump | Foot involvement |
| Skin | Peau d'orange, hyperkeratosis, lichenification |
| Yellow nail syndrome | Primary LE + yellow nails + pleural effusion |

| Complications | Infection/cellulitis (commonest), immunodeficiency, **lymphangiosarcoma** |
|---|---|

| Management | Detail |
|---|---|
| **First line** | **Complex decongestive therapy** (MLD + compression + skin care) |
| Investigation | **Lymphoscintigraphy** |
| Operative (last) | Liposuction, **lymphovenous anastomosis** |
| DDx | Lipedema, venous, systemic pitting edema, Klippel-Trenaunay |

**Bold = high-yield.**
