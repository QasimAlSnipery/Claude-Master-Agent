## Parathyroid Glands — Revision Table

| Basics | Detail |
|---|---|
| Origin | **3rd & 4th pharyngeal pouches** |
| Sensor | Chief cells (low Ca, high PO₄) |
| PTH | Raises Ca, lowers PO₄; t½ 3–5 min |

| Hypoparathyroidism | Detail |
|---|---|
| Causes | **Surgical**, autoimmune, DiGeorge, pseudohypo (Albright) |
| Signs | Tetany, **Chvostek**, **Trousseau** |
| Bloods | ↓Ca, ↓PTH, ↑phosphate |

| Hyperparathyroidism mnemonic | Bones, Stones, Thrones, abdominal Groans, Psychiatric overtones |
|---|---|

| Type | Cause | Ca | PO₄ | PTH |
|---|---|---|---|---|
| **Primary** | **Adenoma >80%** | ↑ | ↓ | ↑ |
| Secondary | CKD / low vit D | ↓ | ↑ | ↑ |
| Tertiary | Autonomous (ESRD) | ↑ | — | ↑ |

| MEN | Components |
|---|---|
| **MEN 1 (PPP)** | Pituitary + Pancreatic + Parathyroid |
| **MEN 2A (PPM)** | Parathyroid + Pheochromocytoma + Medullary thyroid |
| MEN 2B | Pheo + Medullary + mucosal neuromas |

| Bone signs | Osteitis fibrosa cystica: brown tumours, salt-and-pepper skull, distal clavicle tapering |
|---|---|

| Localisation | Note |
|---|---|
| **Sestamibi** | **Most accurate** |
| Ultrasound | Cheap, operator-dependent |
| 4D-CT | Anatomical + functional |

**Bold = high-yield.**
