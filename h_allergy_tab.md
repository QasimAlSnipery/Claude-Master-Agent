## Allergic Diseases — Revision Table

| Mechanism | Detail |
|---|---|
| Type | **Type I, IgE-mediated** |
| Early phase | Mast cell → **histamine**; antihistamine works |
| Late phase | **Th2 + eosinophils**; needs **steroids** |

| Mediator | Action |
|---|---|
| **Histamine** | Bronchoconstriction, vascular leak |
| Leukotrienes (SRS-A) | Sustained bronchoconstriction |
| PAF | Severity marker |

| Atopy | Risk |
|---|---|
| 2 atopic parents | 75% |
| 1 parent | 50% |
| None | 15% |
| Hallmark | **Th2 cells vs allergen** |

| Diagnosis | Detail |
|---|---|
| **Skin-prick test** | Weal size |
| Specific IgE | Serum |
| **Tryptase** | Peaks 1–2h (anaphylaxis) |

| Treatment | Use |
|---|---|
| Avoidance | First line |
| Antihistamine | Immediate (not asthma) |
| Corticosteroid | Late phase |
| Montelukast | Asthma (aspirin-induced) |
| Omalizumab | Anti-IgE, severe asthma |
| Immunotherapy | IgE-mediated (stings, rhinitis) |

| Anaphylaxis | Detail |
|---|---|
| Triggers | **Nuts, shellfish, penicillin, stings** |
| Features | Hypotension, bronchospasm, laryngeal oedema |
| **Treatment** | **IM adrenaline** first + ABC/O₂ |
| Carry | EpiPen 0.3mg adult |

**Bold = high-yield.**
