## Arterial Aneurysms — Revision Table

| Term | Definition |
|---|---|
| Aneurysm | ≥1.5× normal diameter, **all 3 layers** |
| **AAA** | Infrarenal aorta **>3 cm** |
| Ectasia | Enlarged but <50% over normal |
| Arteriomegaly | Diffuse multi-segment >50% |
| **True** | All 3 layers |
| **False/pseudo** | CT/haematoma wall + neck; iatrogenic/anastomotic |

| Shape | Features |
|---|---|
| Fusiform | Uniform, circumferential, atherosclerotic |
| **Saccular** | Focal eccentric; **higher rupture → repair smaller**; renal/cerebral |

| AAA | Detail |
|---|---|
| Frequency | 30% infrarenal; 9× > thoracic; +6%/decade after 65 |
| Top risk | **Smoking** (dose-dependent) |
| Rupture | ↑↑ when **>5 cm**; saccular worse |
| Presentation | Asymptomatic; pulsatile mass (thin) |
| Imaging | US screen, **CTA excellent**, MRA |
| Repair | Open vs **EVAR** |

| Peripheral site | Key numbers / facts |
|---|---|
| **Popliteal (PAA)** | **70% of peripheral**, men 65–80, bilateral 50%, 30–50% +AAA; presents as ischaemia; repair if Sx or **>2cm** |
| Iliac | Common iliac commonest; internal iliac **40% rupture / 31% death**; bilateral 50% |
| Femoral | Complications if **>3.5cm**/thrombus; **88% synchronous aneurysm**; infected IVDU pseudo = most dangerous |
| **Splenic (visceral)** | Commonest visceral **(~60%)**; women, portal HTN; **pregnancy = rupture** |
| Renal | 0.1%; FMD, saccular; **3rd-trimester rupture: 70%/100% mortality** |
| Subclavian | Thoracic outlet repetitive trauma |

| Pseudoaneurysm Rx | US-guided compression / **thrombin injection**; open if infected |
|---|---|

**Bold = high-yield.**
