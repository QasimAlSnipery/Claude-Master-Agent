## HIV — Revision Table

| Virology | Detail |
|---|---|
| Virus | **RNA retrovirus** (lentivirus) |
| Types | HIV-1 (global), HIV-2 (W Africa, milder) |
| Target | **CD4 cells** (gp120 + CCR5/CXCR4) |
| Immune | **CCR5 delta-32** homozygotes |
| Enzymes | Reverse transcriptase, **integrase** (proviral DNA) |

| Transmission risk | % |
|---|---|
| Blood transfusion | **90%** |
| Vaginal delivery | 15% |
| Receptive anal | 1.4% |
| **Needlestick** | **0.3%** |
| Breastfeeding | 0.5%/month |

| Diagnosis | Use |
|---|---|
| Antibody (ELISA) | 2 positive confirm |
| **p24 antigen** | Earlier than antibody |
| **PCR (RNA)** | Infants, primary infection |

| Monitoring | Value |
|---|---|
| CD4 normal | >500 |
| CD4 <200 | **AIDS-defining risk** |
| Viral load | ART response; **undetectable = untransmittable** |

| WHO stage | Feature |
|---|---|
| 1 | Asymptomatic/lymphadenopathy |
| 2 | Weight loss, zoster |
| 3 | Oral candida, hairy leukoplakia, **TB** |
| **4 (AIDS)** | **PCP, Kaposi, toxoplasmosis, wasting** |

| ART class | Example |
|---|---|
| NRTI | Tenofovir, emtricitabine |
| NNRTI | Efavirenz |
| Protease inhibitor | Darunavir |
| **Integrase inhibitor** | Dolutegravir |

| Prevention | Detail |
|---|---|
| **PEP** | Within **72 h** |
| **PrEP** | Daily tenofovir + emtricitabine |

**Bold = high-yield.**
