## Spine (Anatomy / Disc / Infection) — Revision Table

| Anatomy | Detail |
|---|---|
| Vertebrae | 7C, 12T, 5L, 5S, 3–4 coccygeal |
| Curves | Cervical lordosis 40°, thoracic kyphosis 20–40°, lumbar lordosis 30–60° |
| Cervical nerves | **8 nerves**, exit above numbered vertebra; vertebral artery enters C6 |
| Disc | NP (type II, 70–90% water) + annulus (type I, >60 lamellae); **avascular** |
| Key fact | **PLL weak → posterolateral prolapse**; facets carry 18% lumbar load |

| Disc herniation grade | Meaning |
|---|---|
| 1 Bulge | Diffuse |
| 2 Protrusion | Focal, annulus intact |
| 3 Extrusion | Through annulus |
| 4 **Sequestration** | Free fragment |

| Cervical disc prolapse | Detail |
|---|---|
| Who | Male, 30–50, smoker; C5-6, C6-7 |
| Signs | Myelopathy: Babinski, **Hoffman**, gait/dexterity loss |
| Imaging | **MRI diagnostic** |
| Surgery | ACDF, CDR; if acute myelopathy/progressive deficit |

| Lumbar disc prolapse | Detail |
|---|---|
| Where | **L4-5, L5-S1**; 3rd–4th decade |
| Root | Affects root **below** (L4-5 → L5) |
| Worse with | Cough/Valsalva, sitting, driving |
| Tests | SLR, bowstring, femoral stretch |
| Conservative | **90% improve**, 6 wks; treat patient not MRI (30% false +) |
| **Absolute surgery** | **Cauda equina (ASAP)**, progressive motor deficit |

| Pyogenic spinal infection | Detail |
|---|---|
| Levels | **Lumbar 59%** > thoracic 33% > cervical 8% |
| Organism | **S aureus 30–50%** |
| Risk | 7th decade, diabetes, immunosuppression, IVDU |
| Clinical | Pain/tenderness 90%, fever 61% |
| Investigations | **ESR>50 (80–90%)**, CRP, biopsy; **MRI best (>95%)** |
| Treatment | **Abx by sensitivity IV→oral >6 wks**; surgery if abscess/sequestra/deficit/instability |

| Pott's (spinal TB) | Detail |
|---|---|
| Type | Commonest granulomatous |
| **Feared** | **Pott's paraplegia** (early = pressure; late = deformity/vascular) |
| X-ray | 2 adjacent vertebrae osteoporosis + disc narrowing → angular kyphosis |
| Dx | Mantoux/PCR, biopsy |
| Treatment | **Anti-TB chemo** (= surgery efficacy); anterior excision + rib strut graft |

**Bold = high-yield.**
