## Common Knee Problems — Revision Table

| Joint basics | Detail |
|---|---|
| Type | Hinge synovial; **ligaments = main stabiliser** |
| Symptoms | Pain, catching, clicking, **locking** |
| Imaging | **MRI ligaments**, standing AP X-ray for OA |

| Patellar dislocation | Detail |
|---|---|
| Direction | **Lateral** (valgus alignment) |
| Treatment | Conservative 2–3 wk; surgery if MPFL torn |

| Knee dislocation | Detail |
|---|---|
| Ligaments | ACL+PCL+LCL+MCL ruptured |
| **Vascular** | **40% popliteal artery, 20% peroneal nerve** |
| Treatment | Immediate reduction (no hyperextension), ex-fix if vascular |

| Extensor rupture by age | Site |
|---|---|
| Adolescent | Tibial tubercle avulsion |
| Young adult | Patellar tendon rupture |
| Middle-aged | Patella fracture |
| Older/ill | Quadriceps tendon rupture |

| Meniscus | Detail |
|---|---|
| Medial vs lateral | **Medial torn more** (less mobile) |
| Healing | Only **outer third** (vascular) |
| Sign | **Locking = bucket-handle tear** |
| Treatment | Arthroscopic meniscectomy/repair |

| OCD | Detail |
|---|---|
| Who | Male 15–20; trauma |
| **Sign** | **Wilson's sign** |
| Treatment | Stable→rest 6–12mo; unstable→fix/remove |

| Deformity | Detail |
|---|---|
| Normal | **5–7° valgus** |
| Physiological | Bow legs (baby), knock knees (4yr) — reassure |
| Operate | If marked by age 10 |

| Knee OA | Detail |
|---|---|
| Note | **Commonest large-joint OA** |
| Who | >50, overweight, bow-leg |
| X-ray | Narrowing, sclerosis, cysts, osteophytes |
| Treatment | Stick, quads, steroid; osteotomy/replacement |

| Swelling | Cause |
|---|---|
| Acute haemarthrosis | **ACL tear**, haemophilia |
| Septic | Staph (adult: gonococcal); urgent drainage |
| Chronic | OA, RA, PVNS, TB |

**Bold = high-yield.**
