# THE SMALL INTESTINE (Surgery) — Revision Table
Bold = high-yield. Dr Mezjda Rashaan.

## ANATOMY &amp; DIVERTICULA
| Topic | Key points |
|---|---|
| Function/anatomy | B12 absorption; GALT; **ligament of Treitz, valvulae conniventes, Peyer's patches**; **Cajal cells = pacemaker** |
| **Meckel's** | **Rule of 2s** (2%, 2 inches, 2 feet from ICV, M:F equal, &lt;2yrs); **60% gastric mucosa**; vitelline duct; **painless GI bleed in children**; Tc-99m scan; **Littre's hernia**; diverticulectomy if symptomatic |
| Duodenal | Periampullary; cholangitis/pancreatitis/Oddi dysfunction; ERCP problem |
| Jejunal | Connective tissue disease; malabsorption, B12 deficiency |
| Blind loop | Stasis → bacteria → **malabsorption (fat + B12), steatorrhoea** |

## CROHN'S DISEASE
| Aspect | Key points |
|---|---|
| Features | F&gt;M, 15–30; **terminal ileum 60%**, mouth→anus; **smoking ↑3×** |
| Pathology | **Transmural non-caseating granuloma; skip lesions; cobblestone; rectal sparing** |
| Extra-intestinal | **Sclerosing cholangitis, ankylosing spondylitis, erythema nodosum, pyoderma gangrenosum** |
| Investigations | **Barium = string sign of Kantor**; MRI perianal |
| Treatment | **Steroid + 5-ASA + azathioprine**; **surgery for complications only** (obstruction, fistula); stricturoplasty |

## INFECTIONS &amp; FISTULA
| Condition | Key points |
|---|---|
| Amoebiasis | E. histolytica; **bottlenecked ulcer** (sigmoid/rectum); **metronidazole + diloxanide** |
| Intestinal TB | **Ulcerative (2° pulmonary) vs hyperplastic (ileocaecal)**; rarely fistula (unlike Crohn's) |
| Typhoid | Lower ileum longitudinal ulcer; perforation/haemorrhage |
| **Fistula** | **&gt;80% iatrogenic; 90% close in 5 wks**; **FRIEND (Foreign body, Radiation, Inflammation, Epithelialization, Neoplasm, Distal obstruction)** |

## TUMOURS
| Tumour | Key points |
|---|---|
| Overview | **Rare** (1–2% of GI cancers); duodenum &gt; ileum |
| Benign | **Adenoma commonest**; **Peutz-Jeghers (hamartoma + mucocutaneous melanosis)** |
| Adenocarcinoma | 35–50%; risk Crohn's/coeliac/FAP; resection |
| **Carcinoid** | Enterochromaffin; **serotonin/5-HIAA**; **crisis only with metastases (flushing, right heart fibrosis)**; somatostatin |
| Lymphoma | Western (B), **Mediterranean (alpha-chain)**, coeliac (T-cell) |
| **GIST** | **KIT (CD117) mutation, Cajal cells**; mostly stomach; **imatinib** |
