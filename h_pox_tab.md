## Monkeypox & Chickenpox — Revision Table

| Feature | Monkeypox | Chickenpox |
|---|---|---|
| Virus | **Orthopoxvirus (DNA)** | **VZV (herpesvirus)** |
| **Lymphadenopathy** | **Yes (key)** | No |
| Distribution | Whole body, palms/soles | Central/trunk |
| Lesion stages | **All same stage** | **Different stages** |
| Incubation | 6–13 d (5–21) | 10–21 d (avg 14–16) |

| Monkeypox | Detail |
|---|---|
| Zoonotic | Bites, bushmeat, droplets |
| Invasion | Fever + **lymphadenopathy** |
| Rash | Face 95%, palms/soles 75%; macule→crust |
| Diagnosis | **PCR (vesicle swab)** |
| Vaccine | **Smallpox vaccine ~85%** |
| Antiviral | Tecovirimat (ST-246) |

| Chickenpox | Detail |
|---|---|
| Transmission | **Droplet/airborne**, vesicle fluid |
| Infectious | 1–2 d before rash → **all crusted** |
| vs Shingles | Shingles = reactivation, **1 dermatome**, pain |
| Vaccine | **Live**, 12–15 mo + 4–6 yr |
| Post-exposure | Vaccine <72h; **VZIG** high-risk <96h |
| Isolation | Until lesions crust (~7–10 d) |

**Bold = high-yield.**
