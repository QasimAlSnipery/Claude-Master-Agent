## Hydatid Disease — Revision Table

| Basics | Detail |
|---|---|
| Parasite | ***Echinococcus granulosus*** (tapeworm) |
| **Definitive host** | **Dog / carnivores** |
| Intermediate | Sheep, cattle, **humans** |
| Transmission | Swallow eggs (dog faeces) |

| Cyst layers | Outer laminated + inner germinal + protoscolices; daughter cysts |
|---|---|

| Organ involvement | % |
|---|---|
| **Liver** | **>65%** |
| Lung | 25% |
| Other | Brain, bone, eye, heart |

| Clinical | Detail |
|---|---|
| Usually | Asymptomatic |
| Symptoms | Cyst >8–10 cm |
| **Rupture** | **Anaphylaxis**, seeding |

| Diagnosis | Detail |
|---|---|
| **Imaging** | **Ultrasound (preferred)**, CT |
| Blood | Eosinophilia, **ELISA + immunoblot** |
| LFT | Unreliable (ALP may ↑) |

| Treatment | Indication |
|---|---|
| **Surgery** | Large/complicated/superficial/infected |
| PAIR | Inoperable, relapse |
| **Albendazole** | 12–15 mg/kg, <5cm, post-op ≥2yr |
| CI drug | Rupture-prone cysts, early pregnancy |

| Prevention | Deworm dogs, hygiene, gloves |
|---|---|

**Bold = high-yield.**
