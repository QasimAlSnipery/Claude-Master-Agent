# PREOPERATIVE ASSESSMENT &amp; GENERAL ANAESTHESIA (Revision Table)
Bold = high-yield. Amir M. Boujan.

## FASTING &amp; ASA
| Topic | Key points |
|---|---|
| **ASA fasting** | **Clear liquids 2h, breast milk 4h, formula 5–6h, non-human milk 6h, light meal 6h, heavy meal 8h** |
| ASA I–II | I = healthy; II = mild disease (smoker, **pregnancy, obesity 30–40, controlled DM/HTN**) |
| ASA III–IV | III = severe + limitation (BMI≥40, ESRD dialysis, **MI/stent &gt;3mo**); IV = constant threat (**MI/stent &lt;3mo**, shock, sepsis) |
| ASA V–VI / E | V = moribund (ruptured AAA); VI = brain-dead donor; **E = emergency suffix** |

## GENERAL ANAESTHESIA
| Topic | Key points |
|---|---|
| Definition | Drug-induced LOC, not arousable by pain |
| **Components** | **Immobility, amnesia, analgesia, loss of reflexes, no harm** |
| Depth | Minimal → moderate (conscious) → deep → GA (no pain response, airway intervention) |
| Indications | NMB needed, secure airway, not for MAC/regional, uncooperative, pain not amenable to local |

## STEPS OF GA
| Step | Key points |
|---|---|
| **Preoxygenation** | 100% O2 ×3 min (denitrogenation) — apnoea safety margin |
| Induction | IV (**propofol/thiopental/etomidate/ketamine**) commonest adult; inhaled (**sevoflurane**) paediatric |
| **RSI** | Aspiration risk → **preoxygenate + propofol + succinylcholine/rocuronium + CRICOID PRESSURE + no mask ventilation → intubate** |
| Airway | Mask → NMB → laryngoscopy/intubation or supraglottic |
| Maintenance | **Halogenated agents = mainstay**; N2O (adjunct, low potency); propofol (↓PONV); **TIVA for evoked potential monitoring (scoliosis)** |
| Recovery/PACU | Inhaled via lungs; IV metabolised; NMB reversed; PACU monitoring |
