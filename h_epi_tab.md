## Epidemiology of Communicable Diseases — Revision Table

| Concept | Detail |
|---|---|
| Communicable disease | Infectious + person-to-person |
| **Triad** | **Agent + host + environment** |
| Describe by | **Person, place, time** |
| Forms | Subclinical vs clinical |

| Agent must | Multiply → susceptible host → exit → infect another |
|---|---|

| Triad element | Detail |
|---|---|
| Agent | Virus, bacteria, protozoa, helminth, fungus |
| Host | Immunity, nutrition, age, pregnancy, HIV/diabetes |
| Environment | Sanitation, poverty, climate, crowding |

| Reservoir | Human, animal, non-living (soil) |
|---|---|

| **Carrier** | Infected, **no symptoms, transmits** (more dangerous) |
|---|---|
| Examples | Meningitis 90% carriers; cholera |
| Types | Temporary (incubatory, convalescent), **chronic (Salmonella)** |

| Transmission | Detail |
|---|---|
| Contact | Direct (flu/STD) / indirect (fomites) |
| Inhalation | Droplet (direct) / airborne (indirect) |
| Ingestion | Food & water |
| Skin penetration | Vector / wound |
| Transplacental | Mother → fetus |

| Time period | Meaning |
|---|---|
| **Latent** | Infection → **able to transmit** |
| **Incubation** | Infection → **symptoms** |
| Infectious period | When transmits (cholera after sx, measles before) |

| Epidemic type | Detail |
|---|---|
| Point source | Single brief exposure |
| Extended source | Prolonged exposure |
| **Propagated** | Person-to-person waves |

**Bold = high-yield.**
