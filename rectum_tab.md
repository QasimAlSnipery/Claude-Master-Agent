# THE RECTUM (Surgery) — Revision Table
Bold = high-yield. Dr Dara Mohammed.

## CLINICAL &amp; PROLAPSE
| Topic | Key points |
|---|---|
| Symptoms | **Fresh PR bleeding, altered habit, mucus, tenesmus, prolapse, proctalgia**; DRE essential |
| Mucosal prolapse | 1–4cm; adults with **3rd-degree haemorrhoids (mucohaemorrhoidal)**; phenol/banding |
| **Full-thickness (procidentia)** | All layers, &gt;4cm; **women 6×; 50% incontinence**; **abdominal rectopexy (recurrence 10%) vs perineal (Delorme's/Altemeier's, frail)** |
| Proctitis | UC, Crohn's, radiation, **C. diff, amoebic, TB, LGV, bilharzia** |

## POLYPS &amp; CARCINOMA
| Topic | Key points |
|---|---|
| Polyps | **&gt;1cm = 10% malignancy; villous &gt; tubular**; EMR &lt;2cm, ESD &gt;2cm; remove all + colonoscopy |
| CRC overview | **2nd commonest cancer; rectum most frequent site (1/3)**; adenoma-carcinoma sequence (**APC, TP53, KRAS**) |
| Risk | IBD, family history (HNPCC/Lynch 3%, FAP) |
| Clinical | **Bleeding, tenesmus (spurious diarrhoea, bloody slime)**; late = pain/weight loss |
| Diagnosis | **Colonoscopy + biopsy; MRI rectum (local); CEA; CT TAP staging** |
| Spread | **Circumferential local (dictates management)**; lymphatic upward; venous → **liver 34%, lung 22%** |

## STAGING &amp; TREATMENT
| Topic | Key points |
|---|---|
| **Dukes** | **A (wall, &gt;90%), B (extrarectal, 70%), C (nodes, 40%), D (mets)** |
| TNM | T1 submucosa → T4 organs; N1 (1–3), N2 (≥4 nodes); M1 distant |
| Grading | Well/mod/poor; **signet ring = poor**; vascular/perineural invasion |
| Treatment | **MDT; TME (mesorectal excision) + radical resection**; anterior resection/APR/Hartmann's; liver mets resectable |
