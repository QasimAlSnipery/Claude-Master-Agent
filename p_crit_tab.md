## Care of the Critically Ill Patient — Revision Table

| Phase | Note |
|---|---|
| Primary survey | ABCDE — find lethal injuries |
| Resuscitation | **Simultaneous** with primary |
| Secondary survey | Head-to-toe when stable |
| Definitive + tertiary | Final care |

| A — Airway | Detail |
|---|---|
| Assume | **C-spine injury** until cleared |
| Intubate if | Obstruction, **GCS<9**, resp failure |
| Technique | Manual in-line stabilisation, cricoid, bougie |
| Can't intubate | i-Gel → **surgical cricothyroidotomy** |

| B — Breathing emergency | Treatment |
|---|---|
| **Tension pneumothorax** | Decompress 2nd ICS MCL / 5th ICS AAL → drain |
| Open pneumothorax | 3-sided occlusive dressing + drain |
| Massive haemothorax | **>1500mL**; thoracotomy if >200mL/h |
| Cardiac tamponade | FAST; thoracotomy/pericardiocentesis |

| C — Circulation | Detail |
|---|---|
| ↓Systolic BP | >30% loss (~1500mL) |
| ↓Consciousness | 40–50% loss |
| Access | 2× 14G IV / intraosseous |
| **Permissive hypotension** | 250mL boluses, systolic **~80mmHg** |
| Blood | **1:1:1** RBC:FFP:platelet |
| **TXA** | 1g within **3 h** |

| D / E | Detail |
|---|---|
| Disability | Pupils + GCS |
| Exposure | Undress, **prevent hypothermia** |
| Hypothermia | <35°C = mortality predictor |

| Tubes / imaging | Detail |
|---|---|
| Catheter | Output = resus marker |
| Gastric tube | Orogastric if basal skull # |
| **Whole-body CT** | Within 30 min |

**Bold = high-yield.**
