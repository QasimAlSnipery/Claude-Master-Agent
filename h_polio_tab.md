## Poliomyelitis — Revision Table

| Virus | Detail |
|---|---|
| Family | **Picornavirus / Enterovirus** (RNA) |
| Serotypes | **3** (type 2 eradicated 1999, type 3 2020) |
| Endemic | Pakistan, Afghanistan (type 1) |
| Reservoir | **Human only** |
| Transmission | **Faecal-oral** |

| Clinical | % |
|---|---|
| Asymptomatic/mild | **~95%** |
| Aseptic meningitis | 1–2% |
| **Paralytic** | **<1%** |
| Paralysis | Asymmetric flaccid, **legs>arms**, permanent |
| CFR (paralytic) | 2–10% |
| Precipitants | Trauma, IM injection, tonsillectomy, pregnancy |

| Diagnosis | Stool (72h, persists 3–6 wk), throat (36h) |
|---|---|

| Vaccine | Detail |
|---|---|
| **OPV (Sabin live)** | Intestinal + circulating; **CI immunocompromised**; rare VAPP |
| **IPV (Salk inactivated)** | Circulating Ab; Iraq 4 & 6 mo |

| Eradication pillars | |
|---|---|
| 1 | Routine immunisation |
| 2 | **NIDs** (all <5) |
| 3 | Mopping up |
| 4 | **AFP surveillance** |

| AFP surveillance | Target |
|---|---|
| Report | All AFP **<15 yrs within 48 h** |
| Specimens | **2 stools, 24–48h apart, within 14 d** |
| Quality | ≥2 non-polio AFP /100,000 |
| Polio-free | 3 yrs no wild virus |

**Bold = high-yield.**
