## RTIs & STIs — Revision Table

| RTI cause | Examples |
|---|---|
| Endogenous (commonest) | Candidiasis, bacterial vaginosis |
| Iatrogenic | IUD, abortion, catheter |
| STI bacterial | Chlamydia, gonorrhoea, syphilis |
| STI viral (**4 H's**) | **HIV, HPV, Herpes, Hepatitis** |
| STI parasitic | Trichomoniasis |

| PID | Detail |
|---|---|
| Cause | Ascending gonorrhoea/chlamydia |
| Complications | **Infertility, ectopic, chronic pain** |

| STI | Organism / sign | Treatment |
|---|---|---|
| **Trichomoniasis** | T. vaginalis; **strawberry cervix**, frothy green | **Metronidazole** |
| **Chlamydia** | C. trachomatis; often asymptomatic | **Doxycycline 7d** |
| **Gonorrhoea** | N. gonorrhoeae; G-ve diplococci | **Ceftriaxone IM** |
| **Syphilis** | T. pallidum; **painless chancre** | **Benzathine penicillin** |
| HPV | 16/18 → cancer, 6/11 → warts | Vaccine, cryo |
| **Herpes** | HSV-2; vesicles/ulcers | **Aciclovir** |

| Diagnosis | Note |
|---|---|
| **NAAT** | Chlamydia, gonorrhoea, trich, herpes |
| Gonorrhoea | + Gram stain (intracellular diplococci) |
| Syphilis | **VDRL/RPR (screen) + TPHA (confirm)** |

| Prevention | Condoms, monogamy, **HPV/HBV vaccine**, screening, treat partners |
|---|---|

**Bold = high-yield.**
