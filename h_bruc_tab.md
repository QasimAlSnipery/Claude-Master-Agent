## Brucellosis — Revision Table

| Organism | Detail |
|---|---|
| Bacteria | **Brucella** — G-ve coccobacilli |
| Nature | **Facultative intracellular** (macrophages) |
| Growth | Slow (14d–4wk), enriched media |

| Species | Animal |
|---|---|
| **B. melitensis** | Sheep/goats (most virulent) |
| B. abortus | Cattle |
| B. suis / canis | Pigs / dogs |

| Transmission | Route |
|---|---|
| Dairy | **Unpasteurised milk/cheese** |
| Contact | Animal fluids/placenta via cuts |
| Inhalation | Aerosols (abattoir/lab) |
| Occupational | Vets, farmers, butchers |

| Clinical | Feature |
|---|---|
| Incubation | 5–30 d (up to 6 mo) |
| Fever | **Undulant** (rise/fall) + sweats |
| Bone/joint | **Spondylitis, sacroiliitis, arthritis** |
| Serious | Endocarditis |
| CFR | <2% |

| Diagnosis | Detail |
|---|---|
| Definitive | **Culture** (blood/bone marrow, slow) |
| Serology | **Rose Bengal, ELISA**; IgM→IgG |
| Molecular | PCR |

| Treatment | **Doxycycline 6wk + rifampicin or streptomycin** |
|---|---|

| Prevention | Pasteurise milk, animal vaccination, PPE |
|---|---|

**Bold = high-yield.**
