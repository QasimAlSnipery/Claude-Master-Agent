## Carcinoma of the Pancreas — Revision Table

| Feature | Detail |
|---|---|
| Commonest type | **Ductal adenocarcinoma (>85%)**, head of gland |
| Top modifiable risk | **Smoking** |
| Genetic risk | BRCA2, hereditary pancreatitis (50–70×), Lynch, Peutz-Jeghers, FAP |
| Hallmark presentation | **Painless obstructive jaundice** |
| Courvoisier's law | Palpable GB + jaundice → malignancy not stones |
| Red flags | New diabetic >50, unexplained pancreatitis, back pain |

| Cystic tumour | Behaviour |
|---|---|
| Serous cystadenoma | Older women, **benign** |
| Mucinous (MCN, IPMN) | **Malignant potential** |
| Periampullary | ⅓ of head tumours, better prognosis |

| Investigation | Role |
|---|---|
| US | Bile duct dilated? |
| **CT (contrast)** | Staging test of choice |
| MRCP | "Double duct sign" |
| ERCP + stent | Cholangitis, doubt, delay to surgery |
| EUS + FNA | Tissue dx (no peritoneal spillage) |
| CA19-9 | Baseline / recurrence marker |

| Unresectable if | Resectable despite |
|---|---|
| Liver/peritoneal mets | Tumour size |
| Distant nodes | Duodenum/stomach/colon invasion |
| **SMA/hepatic/coeliac encasement** | Local field nodes |

| Surgery | Indication |
|---|---|
| **PPPD** | Head/ampulla (standard) |
| Whipple (+antrum) | FAP, duodenal encroachment |
| Total pancreatectomy | Multifocal / main-duct IPMN |
| Distal pancreatectomy + splenectomy | Body/tail |

| Outcomes | Value |
|---|---|
| Resectable at dx | <15% |
| PPPD mortality | 3–5% |
| Morbidity | 30–40% |
| Pancreatic leak | ≥10% (↓ with octreotide) |
| Adjuvant | Gemcitabine ± 5-FU 6 mo |

| Palliation | Method |
|---|---|
| Jaundice | ERCP stent / percutaneous / bypass |
| Duodenal obstruction (15%) | Gastrojejunostomy / duodenal stent |
| Pain | Analgesia + **coeliac plexus block** |

**Bold = high-yield.**
